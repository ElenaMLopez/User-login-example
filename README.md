## Project setup

```
npm install
```

### Compiles and hot-reloads for development

To run te server with hot-reload just open a terminal and type:

```
npm run serve
```

Then you can visit `http://localhost:8080/` and see the result.

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```
